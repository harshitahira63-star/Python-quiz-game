🧠 Python Quiz Game

A simple command-line quiz game built using Python.
This program asks the user basic computer-related questions and checks whether the answers are correct or not.

The project demonstrates basic Python concepts such as user input, conditional statements, and program flow control.


---

🚀 Features

Interactive command line quiz

Multiple computer-related questions

Instant feedback (Correct / Incorrect)

Beginner-friendly Python project

Uses basic if-else logic



---

🛠️ Technologies Used

Python

CLI (Command Line Interface)



---

📂 Project Structure

QuizGame.py

This file contains the complete logic of the quiz game.


---

▶️ How to Run the Program

1. Install Python on your system.


2. Download or clone this repository.


3. Open the project folder in VS Code or terminal.


4. Run the program:



python QuizGame.py
